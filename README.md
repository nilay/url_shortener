# URL Shortener

Small Ruby on Rails application to shorten the url

Things you may want to cover:

* Ruby version: 2.3.1


### Get shortended url
```
POST /
param: url
response (in text format): shortended url
```


### use shortended url to redirect to original url
```
GET <shortended url>`
it will redirect to original url
```

## Run Unit Test
`$ rspec`
